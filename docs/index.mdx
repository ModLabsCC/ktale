## What KTale is

KTale is a **speculative** Kotlin server SDK foundation for Hytale. The Hytale server API does not exist (for us) yet, so KTale is designed around:

- **Adapter-based architecture**: stable core + replaceable platform adapters
- **Capabilities over inheritance**: small, composable interfaces
- **Testability without a real server**: a deterministic fake platform
- **Honest uncertainty**: anything speculative is isolated and removable
