KTale is a lightweight Kotlin-first helper library for building **Hytale Server plugins**.

This `/docs` folder is for human-written guides (MDX/Markdown).

API reference is generated from KDoc via **Dokka** and published to **GitHub Pages** by `.github/workflows/docs.yml`.
